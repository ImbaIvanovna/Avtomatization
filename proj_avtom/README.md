# Музыкальный магазин

Веб-приложение для управления базой данных музыкального магазина, содержащей информацию о музыкантах, ансамблях, музыкальных произведениях и пластинках.

## Описание

Приложение поддерживает следующий функционал:

1. **Количество музыкальных произведений заданного ансамбля** - просмотр всех произведений, записанных конкретным ансамблем
2. **Названия всех компакт-дисков заданного ансамбля** - список всех пластинок с записями ансамбля
3. **Лидеры продаж текущего года** - топ-10 самых популярных компакт-дисков
4. **Управление данными о компакт-дисках** - добавление, редактирование и удаление пластинок
5. **Управление данными об ансамблях** - добавление, редактирование и удаление ансамблей

## Технологии

- **Backend**: Python Flask
- **Frontend**: HTML, CSS, JavaScript
- **База данных**: SQLite
- **Стили**: Современный CSS с градиентами и анимациями

## Структура базы данных

### Основные таблицы:
- `musicians` - музыканты (исполнители, композиторы, дирижеры)
- `ensembles` - ансамбли (оркестры, группы, квартеты и т.д.)
- `compositions` - музыкальные произведения
- `performances` - исполнения произведений ансамблями
- `records` - пластинки/компакт-диски
- `companies` - компании-производители
- `ensemble_members` - участники ансамблей
- `record_tracks` - записи на пластинках

## Установка и запуск

### Требования
- Python 3.7+
- pip

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Инициализация базы данных
```bash
python database.py
```

### Запуск приложения
```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Использование

1. **Главная страница** - обзор статистики и доступных функций
2. **Произведения ансамбля** - выбор ансамбля для просмотра его произведений
3. **Диски ансамбля** - выбор ансамбля для просмотра его пластинок
4. **Лидеры продаж** - просмотр самых популярных дисков текущего года
5. **Управление дисками** - CRUD операции с пластинками
6. **Управление ансамблями** - CRUD операции с ансамблями
