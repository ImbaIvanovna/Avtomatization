# Схема базы данных

Эта директория предназначена для хранения актуальной схемы базы данных проекта.

## Содержимое

Здесь можно хранить:
- SQL файлы с определением схемы (CREATE TABLE, CREATE INDEX и т.д.)
- Экспортированные схемы базы данных
- Миграции схемы
- Документацию по структуре базы данных

## Формат файлов

Рекомендуется использовать следующие соглашения об именовании:
- `schema.sql` - полная схема базы данных
- `schema_YYYYMMDD.sql` - версионированные схемы
- `migrations/` - директория для миграций (если используется)
